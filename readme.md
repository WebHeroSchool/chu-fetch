Для работы нужно в конец строки поиска вставить ?username= со значением ника пользователя GitHub,
например 6thSence, ChuVo, json, чтобы строка выглядела так: 
	-html?username=6thSence
	-html?username=ChuVo
	-html?username=json
Затем нужно нажать ENTER и запустить функцию вывода данных - "кнопка get data"
